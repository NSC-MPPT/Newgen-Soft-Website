本站KMS服务器，长期维护，服务器来自微软云，真正官方jh。

手动jh请查看：https://www.ghpym.com/kms.html，欢迎更多的网友使用本站服务器制作jh软件。

脚本jh以后，系统会自动每周验证，虽然显示jh180天，但是到了时间，系统会自动续期，无需手动操作，这个续期的功能是Windows自带的，无需手动操作，所以相当于一次jh，永久使用。（果核剥壳）
